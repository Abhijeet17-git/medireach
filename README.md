# MediReach 🏥
AI-powered emergency medical response platform. Real-time hospital bed availability, Gemini AI hospital selection, SOS ambulance dispatch, and hospital verification system. Built for Google Solution Challenge 2026.

## 🌐 Live Demo
- **Frontend:** https://medireach-vert.vercel.app
- **Backend API:** https://medireach-production-9d50.up.railway.app

## 🚀 Features
- Real-time hospital bed availability
- Gemini AI powered hospital selection
- SOS ambulance dispatch
- Hospital verification system
- Role-based access (Super Admin, Hospital Admin, User)
- Live bed count simulation

## 🛠️ Tech Stack
- **Frontend:** React.js (Vercel)
- **Backend:** Spring Boot (Railway)
- **Database:** MySQL (Railway)
- **AI:** Google Gemini API
- **Payment:** Razorpay
